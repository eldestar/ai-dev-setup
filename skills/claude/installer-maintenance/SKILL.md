# Installer Maintenance

Use this skill when changing installer behavior.

1. Inspect the relevant entry point and platform helper.
2. Keep reusable logic in `scripts/common/`.
3. Add or update docs when behavior changes.
4. Run dry-run validation.
5. Summarize changed paths, validation, and remaining platform risks.
