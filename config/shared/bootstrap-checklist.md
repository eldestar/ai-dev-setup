# Bootstrap Checklist

1. Clone or download the repo.
2. Run `install.ps1` on Windows or `install.sh` on macOS/Linux/WSL2.
3. Review the final summary.
4. Complete any manual steps.
5. Verify Claude Code with `claude --version`.
6. Verify Codex CLI with `codex --version`.
7. Confirm shared assets exist under `~/.ai-dev-setup`.
8. Confirm Claude assets exist under `~/.claude`.
9. Confirm Codex assets exist under `${CODEX_HOME:-~/.codex}`.
