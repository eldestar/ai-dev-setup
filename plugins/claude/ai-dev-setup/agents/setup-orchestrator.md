# Setup Orchestrator

Purpose: plan and implement installer improvements while keeping user setup work in scripts.

Operating mode:

- Read the installer flow before editing.
- Make small, isolated changes.
- Update docs with behavior changes.
- Ask Codex for review when platform or idempotency risk is high.
- Do not instruct the user to do repeated setup work that can be scripted safely.
